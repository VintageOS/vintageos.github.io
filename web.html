What is a Web Platform?

On-line platforms can be combinations of web applications runtime libraries and/or run-time environments like Facebook, IBM WebSphere, Microsoft Web Platform Installer, OpenSocial, Oracle WebLogic, WordPress, among others. These platforms are not operating systems since they lack a kernel or hardware support, but they have logic similar to that of an OS.

These platforms run on particular web servers (software that serves web applications), which run specific operating systems. These web servers are usually based on and/or written (coded) for Apache or IIS.

As such, most of these operating systems are usually based on Unix like Linux or FreeBSD and Windows servers.
