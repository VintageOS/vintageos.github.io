What is Assembler?

As a matter of fact, the development of hobby OSs has caught my attention and has started my curiosity of learning assembler, but I have no idea which assembler dialect would be best (easiest) for me. In any case, I have opted for NASM because of the amount of information available on-line related to Linux and easy installation on Cygwin for quick and easy cross-reference (emphasis on easy) — instead of GASM included in the GNU Compiler Collection (GCC), which is included in most Linux distros.

  • A86 (same developer as A386, Eric Isaacson) — MS-DOS
  • A386 (same developer as A86, Eric Isaacson) — Windows
  • flat assembler (fasm), "intentionally stylized with lowercase letters" as explained in its website (Oct. 2022) — DOS, Linux, Unix and Windows
  • GNU Assembler (gasm, gas or as) — DOS, FreeBSD, Linux, OS/2, Windows, etc. (cross-platform)
  • High Level Assembly (High Level Assembly) — FreeBSD, Linux, macOS, Windows
  • Japheth Assembler (JWasm) — AVX (Advanced Vector Extensions), DOS, FreeBSD, Linux, OS/2 and Windows
  • microassembler (UASM or µASM) — for Linux, macOS and Windows
  • Microsoft Macro Assembler (MASM) — for Windows only
  • Netwide Assembler (NASM) — Linux
  • Volker's Assembler (vasm) — needs to be compiled by user for
    • AmigaOS M68k and AmigaOS 4 (latest version)
    • Atari TOS M68k, Atari MiNT and derivatives including EmuTOS
    • Cygwin
    • Haiku
    • MorphOS
    • PowerUp M68k/PowerPC accelerator boards
    • WarpOS (not OS/2 Warp)
    • Windows 32-bit
  • Open Watcom Assembler (WASM) — for DOS, Linux, OS/2 and Windows (not WebAssembly, also abbreviated as WASM)
  • Turbo Assembler (TASM) — MS-DOS and Windows
  • Yasm Modular Assembler (Yasm) — for Cygwin, DOS and Windows
  • or some other variant of assembler either for Linux, Windows or some other OS
